Trilha CSS da DIO – Desafio 01

Criação e desenvolvimento de um projeto prático como parte do bootcamp Trilha CSS da DIO. O objetivo do desafio foi aplicar conceitos de HTML e CSS para construir uma página web profissional, explorando layouts, tipografia, cores, responsividade e elementos interativos.

O projeto resultou em uma página com seções de apresentação, módulos de aprendizado, depoimentos de profissionais e um rodapé informativo. A página foi desenvolvida de forma limpa e organizada, respeitando boas práticas de semântica e acessibilidade.

Ferramentas utilizadas:

• Visual Studio Code – Edição e organização do código.

• HTML & CSS – Estrutura e estilização da página.

• Figma – Referência do protótipo e guia de design.

• Git & GitHub – Versionamento do projeto e publicação do repositório.

Este projeto é um exercício prático que demonstra domínio de conceitos fundamentais de desenvolvimento front-end, incluindo criação de layouts responsivos, interação visual e estética aplicada à web, servindo como material de portfólio para futuros projetos.
